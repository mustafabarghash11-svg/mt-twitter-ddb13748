
# MT TWITTER - خطة بناء المشروع

## نظرة عامة
موقع تواصل اجتماعي شبيه بتويتر باسم **MT TWITTER** بألوان برتقالية وسوداء، مع نظام كامل للتسجيل والتغريدات والرسائل الخاصة.

## التصميم
- **الألوان**: برتقالي كلون أساسي، أسود كلون خلفية/ثانوي، مع تدرجات برتقالية
- **الخط**: عصري وواضح
- **التخطيط**: شريط جانبي للتنقل (الرئيسية، البروفايل، الرسائل) + منطقة المحتوى الرئيسية

## الميزات

### 1. نظام المصادقة (Lovable Cloud)
- صفحة تسجيل دخول / إنشاء حساب
- الحقول: اسم المستخدم، كلمة المرور، اسم العرض، يوزر الديسكورد
- جدول `profiles` مرتبط بـ auth.users

### 2. البروفايل
- عرض وتعديل: اسم العرض، اسم المستخدم، يوزر الديسكورد
- تغيير كلمة المرور
- رفع وتغيير صورة البروفايل (Supabase Storage)

### 3. الصفحة الرئيسية (التغريدات)
- مربع كتابة تغريدة جديدة
- عرض التغريدات (feed) من جميع المستخدمين بترتيب زمني
- كل تغريدة تعرض: اسم المستخدم، الصورة، النص، وقت النشر

### 4. التفاعل مع التغريدات
- إعجاب / إلغاء إعجاب
- التعليق على التغريدات
- تعديل التغريدة (لصاحبها فقط)
- حذف التغريدة (لصاحبها فقط)

### 5. الرسائل الخاصة
- البحث عن مستخدم بالاسم
- بدء محادثة جديدة
- إرسال واستقبال رسائل في الوقت الفعلي (Supabase Realtime)
- قائمة المحادثات مع آخر رسالة

## قاعدة البيانات (Lovable Cloud)
- `profiles` - بيانات المستخدمين
- `tweets` - التغريدات
- `tweet_likes` - الإعجابات
- `tweet_comments` - التعليقات
- `conversations` + `conversation_participants` - المحادثات
- `messages` - الرسائل الخاصة
- Storage bucket للصور الشخصية

## الصفحات
- `/auth` - تسجيل الدخول / إنشاء حساب
- `/` - الصفحة الرئيسية (التغريدات)
- `/profile` - البروفايل
- `/messages` - الرسائل الخاصة
